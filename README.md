Customer Purchase Prediction (Social Ads)
A machine learning project that predicts whether a customer will purchase a product based on social media advertisement data using Logistic Regression.

Overview
This project helps businesses identify potential customers who are more likely to purchase products through social media ads, enabling better targeting and improved marketing efficiency.

Features

Predicts customer purchase behavior
1. Uses Logistic Regression (classification)
2. Data visualization for insights
3. Data preprocessing and feature scaling
4. Improves ad targeting strategy

How It Works
1. Load dataset containing customer details (Age, Estimated Salary, etc.)
2. Preprocess data (cleaning, encoding, scaling)
3. Split data into training and testing sets
4. Train Logistic Regression model
5. Predict whether a customer will purchase or not
6. Evaluate model performance using accuracy and confusion matrix
